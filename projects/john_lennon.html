John Lennon Portfolio

This project is a website about John Lennon, fully written in HTML and CSS. It was made for a school Project. It was actually one of my very first websites.
There is no Javascript at all, which means that it is fully static. Even all of the animations were done in CSS
The website is also fully responsive. I implemented that with simple CSS media-queries that change the structure of the page according to the width of the users screen.
